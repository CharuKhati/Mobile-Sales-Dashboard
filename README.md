# 📱 Mobile Sales Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive **Mobile Sales Dashboard** built in **Microsoft Power BI**. The dashboard provides valuable insights into mobile sales performance, customer behavior, product performance, and transaction trends using interactive visualizations and KPI metrics.

---

## 🎯 Objective

The goal of this dashboard is to help businesses monitor sales performance, identify top-selling products, analyze customer preferences, and support data-driven decision-making.

---

## 📊 Dashboard Features

- 📈 Total Sales KPI
- 📦 Total Quantity Sold
- 💳 Total Transactions
- 💰 Average Sales Value
- 🗺️ Sales by City (Map Visualization)
- 📅 Monthly Sales Trend
- ⭐ Customer Ratings Analysis
- 💳 Transactions by Payment Method
- 📱 Top 3 Mobile Models by Sales
- 📅 Sales by Day of Week
- 📋 Brand-wise Sales Summary
- 🎛️ Interactive Filters
  - Mobile Model
  - Brand
  - Payment Method
  - Month
  - Day Name
